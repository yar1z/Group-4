Thank you for choosing this professional UI Kit! 
This system is designed to work seamlessly in both 2D and 3D environments.

[WHAT'S NEW - v1.1]
- Smart Day/Night System: Now supports 3D Directional Light intensity 
  alongside 2D Camera background colors.

[QUICK START]
1. Drag the "Main_UI_Canvas" prefab into your scene.
2. In your "GameManager" inspector:
   - For 2D: Leave the "World Light" slot empty.
   - For 3D: Drag your "Directional Light" into the "World Light" slot.
3. Ensure an "EventSystem" exists in your hierarchy.

[FEATURES]
- Hybrid Day/Night System (Auto-detects 2D/3D light).
- Optimized Gas & Brake Pedal logic (Mobile Touch Ready).
- Complete Game State Management (Pause, Resume, Audio Mute).
- Dynamic HUD Elements (Score, Fuel, and Cash).

[DEPENDENCIES]
- TextMesh Pro: Required for all UI labels. If fonts are missing, 
  please go to Window > TextMeshPro > Import TMP Essential Resources.
